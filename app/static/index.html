Food Truck Finder API

This API was built by skeller88 and uses the Mobile Food Facility Permit dataset from Data SF. Check out the repo here.

It provides the ability to retrieve all of the food trucks that may be within a certain radius of a particular coordinate.

Open up devtools to see your current coordinates. You can use those coordinates to query the API, or choose your own!

GET /foodtrucks

Returns the closest food trucks to a given latitude and longitude, in order of proximity.

Resource URL

https://sf-food-truck-finder.herokuapp.com/foodtrucks

Parameters

Latitude

Latitude coordinate for which nearby food trucks are found.

Example Values: 37.12341,44.5

Longitude

Longitude coordinate for which nearby food trucks are found.

Example Values: -122.51234,-121

Limit Optional

Maximum number of results to return. Defaults to 10. Must be > 0.

Example Values: 10,1000,45

Within Optional

Maximum distance from coordinate to search, in miles.Defaults to 1 mile. Must be > 0.

Example Values: 10,0.5,2.7

Sample Request

                    
http://sf-food-truck-finder.herokuapp.com/foodtrucks?
latitude=37.54123&longitude=-122.12343&limit=30
